Como rodar:

1 - Crie uma pasta com o nome "calculadorateste"

2 - acesse o terminal

3 - digite cd calculadorateste

4 - digite npm init -y

5 - digite npm install express

6 - criar os arquivos da mesma forma que estão organizados aqui
calculadorateste/
├── public/
│   ├── index.html
│   ├── styles.css
│   └── script.js
├── server.js
